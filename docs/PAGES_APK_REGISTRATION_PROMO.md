# Страницы /apk/, /registration/, /promo-code/

## Шаблоны в теме

- **page-apk.php** — шаблон для страницы «APK / Download App» (slug: `apk`)
- **page-registration.php** — шаблон для страницы «Registration» (slug: `registration`)
- **page-promo-code.php** — шаблон для страницы «Promo Code» (slug: `promo-code`)

Контент в шаблонах выводится через `get_translation()`; при необходимости добавьте переводы в `functions.php` (массив переводов).

## Как сделать страницы в WordPress

1. В админке: **Страницы → Добавить новую**.
2. Создайте три страницы:
   - **Заголовок:** например «تحميل التطبيق (APK)» или «Download APK»  
     **Ярлык (slug):** `apk`  
     **Шаблон:** «APK / Download App» (если виден в выпадающем списке).
   - **Заголовок:** например «التسجيل» или «Registration»  
     **Ярлык:** `registration`  
     **Шаблон:** «Registration».
   - **Заголовок:** например «الرمز الترويجي» или «Promo Code»  
     **Ярлык:** `promo-code`  
     **Шаблон:** «Promo Code».
3. Опубликуйте страницы.

WordPress автоматически подхватит шаблон по slug (`page-apk.php` для страницы с slug `apk` и т.д.), даже если шаблон не выбран вручную.
